Documentation of the SFS Toolbox
================================

This repository provides the sphinx sources for building the online
documentation.

## Usage

To compile it you have to run the following commands:

```bash
$ cd theory
$ make dirhtml
$ cd ../matlab
$ make dirhtml
$ cd ../pyhton
$ make dirhtml
$ cd ../code_references
$ make dirhtml
```
